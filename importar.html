📊 Importação em Lote por Planilha Excel
1
Baixe o modelo de planilha
O modelo já contém exemplos de todos os tipos de questão e uma aba de instruções detalhadas com a descrição de cada coluna.
2
Preencha de cima para baixo — um item por linha
Para questões avulsas, use QUESTAO_C (múltipla escolha) ou QUESTAO_A (Certo/Errado).
Para blocos PAS, crie uma linha PAS_BLOCO com o texto-base e adicione logo abaixo as linhas dos itens (PAS_A, PAS_C, PAS_B, PAS_D). O bloco se encerra quando aparecer outra linha PAS_BLOCO ou uma questão avulsa.
Coluna TIPOO que representaGabarito esperado
QUESTAO_C Questão múltipla escolha (5 alternativas) A / B / C / D / E
QUESTAO_A Questão Certo/Errado CERTO / ERRADO
PAS_BLOCO Início de bloco PAS — define o texto-base compartilhado sem gabarito
PAS_A ↳ Item Tipo A — Certo/Errado CERTO / ERRADO
PAS_C ↳ Item Tipo C — 4 alternativas (A–D) A / B / C / D
PAS_B ↳ Item Tipo B — Numérico (Centena-Dezena-Unidade) ex: 0-1-5
PAS_D ↳ Item Tipo D — Resposta aberta / discursiva texto livre
3
